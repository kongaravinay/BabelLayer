# Contributing to BabelLayer

Thanks for wanting to help out! I built BabelLayer for my own needs, but I'm open to suggestions and improvements.

## How to contribute

1.  **Found a bug?** Open an issue on GitHub. Tell me what happened and how to reproduce it.
2.  **Suggesting a feature?** Also open an issue. Let's talk about it.
3.  **Sending a PR?** Make sure it's clean and doesn't break the tests. Also, stay consistent with the existing naming style.

## Style

- Use descriptive variable and function names.
- Keep functions small and focused on one thing.
- Don't overdo it with comments. If the code is clear, it'll speak for itself.
