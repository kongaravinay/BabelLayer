# Frequently Asked Questions (FAQ)

### 1. What is BabelLayer?
BabelLayer is a specialized tool for mapping and transforming data between different datasets. It's particularly useful for projects where data formats and schemas change frequently.

### 2. Is it free?
Yes! BabelLayer is open-source under the MIT License.

### 3. How do I get started?
Please refer to the [Quick Start](#-quick-start) in the `README.md` or our [新手指南 (Beginner Guide)](docs/BEGINNER_GUIDE.md).

### 4. Which file formats are supported?
Currently, we support **CSV**, **JSON**, **XML**, and **Excel** (.xlsx).

### 5. Can I use it for large datasets?
We primarily focus on desktop-scale datasets. For extremely large datasets, performance may vary depending on your system's memory and Python environment.

### 6. Is it safe to use for sensitive data?
While we provide role-based authentication and local storage, you should always handle sensitive data according to your organization's security policies.

### 7. How can I contribute?
Check out our [Contributing Guide](CONTRIBUTING.md) for more information.
