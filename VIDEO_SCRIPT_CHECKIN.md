# Capstone Check-In Video Script
Target: about 2 to 3 minutes

## Script

Good morning Professor and classmates. My name is [Your Name], and for my capstone I am considering two projects.

The first option is SecureEntry, which is the intelligent badge-access and security dashboard project from the course list.

The second option is BabelLayer, which is my own proposed project focused on data integration and data quality.

Between these two, I am leaning toward BabelLayer.

The reason is that BabelLayer addresses a practical problem that many teams face: data comes from different systems in different formats, with different field names, and that makes integration slow and error-prone.

My proposed solution is a desktop application that guides users through one complete workflow:
login, upload source files, review mapping suggestions, run quality checks, transform the data, and generate a final report.

I plan to include a few core technical components:

1. A desktop interface using PyQt6 for a clear step-by-step workflow.
2. Secure user access with password hashing and role-based permissions.
3. Data ingestion for CSV, JSON, XML, and Excel files.
4. A SQLite database with SQLAlchemy for persistence.
5. Lightweight intelligence features such as schema matching suggestions, anomaly detection, and explanation support.
6. Reporting with charts and PDF output for stakeholders.
7. External connectivity through APIs such as Google Drive and REST.

This is the direction I am most interested in because it combines design, usability, and technical problem-solving in one artifact.

The key challenges I expect are:

1. Improving mapping accuracy when field names are inconsistent.
2. Handling many file structures without breaking the workflow.
3. Keeping outputs understandable and trustworthy for users.
4. Balancing feature scope with a clean and simple user experience.

So in summary, I considered SecureEntry and BabelLayer, and I am most leaning toward BabelLayer as my capstone direction because it solves a real problem and fits the build-and-evaluate goals of this course.

Thank you.

## Quick Delivery Notes

1. Speak slowly and pause after each section.
2. Keep eye contact with the camera for the opening and closing lines.
3. Mention both projects clearly before stating your preferred one.
4. Keep the tone professional and proposal-focused.
